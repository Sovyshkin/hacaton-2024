Инструкция для развертывания на Windows

Для развертывания проекта вам потребуется:

    Инструкция запуска frontend части сайта:

        1. Скачать и установить Python версии 3.12 (https://www.python.org/downloads/release/python-3120/);
        2. Скачать и установить Node.js c Chocolatey (https://nodejs.org/en);
        3. При необходимости перейдите в каталог проекта (cd /hac);
        4. В терминале выполните команду npm install (для установки пакетов);
        5. В терминале выполните команду npm run build (для создания сборки);
        6. В терминале выполните команду npm run serve (это запустит frontend часть и в терминале вы увидете 
        ссылку на которой расположен сайт *пример ссылки: http://localhost:8080/ ).


    Инструкция запуска backend части сайта:

        1. Добавьте новый терминал для этого проекта (для запуска backend части);
        2. При необходимости перейдите в каталог проекта (cd /hac);
        3. В терминале выполните команду pip install -r requirements.txt (для установки пакетов);

        *Если в папке "instance" уже есть база данных с названием "userInfo" пропустите 4 пункт.
        *4. В терминале выполните команду python dbcreate.py или python3 dbcreate.py (для создания базы данных проекта);
        5. В терминале выполните команду python main.py или python3 main.py (это запустит backend часть);
        6. Вы можете открыть сайт по ссылки из пункта 6 из "Инструкция запуска frontend части сайта" и начать тестирование сайта.
